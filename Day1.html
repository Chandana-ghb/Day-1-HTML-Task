Formating Tags :

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web.It provides the basic building blocks for web pages by using a series of elements, called tags,to define different types of content like headings, paragraphs, links, images, lists, and more. HTML tells the web browser how to display text and media on a webpage.

HTML documents start with a declaration that tells the browser to use the latest HTML version. Inside the document, the html element wraps all the content. The head section contains meta-information like the page title, character encoding, and links to stylesheets or scripts. The body section contains all the visible content like text, images, videos, and interactive elements.
